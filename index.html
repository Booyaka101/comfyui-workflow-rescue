ComfyUI Workflow Rescue

Pulls the embedded workflow JSON out of MP4 / MOV / WebM / MKV files exported by ComfyUI-VideoHelperSuite. Everything runs locally in your browser — files never leave your computer.

What this is for

If you have an MP4 or WebM video that was made with ComfyUI's Video Helper Suite and the workflow no longer drag-drops into ComfyUI, the workflow data might still be embedded in the file — just in a shape the ComfyUI frontend can't read directly. This page extracts it and gives you a workflow.json file you can open with ComfyUI's normal Load button.

If you get "no workflow found"

For videos that include audio, the workflow may have been stripped at write time (see VHS PR #653). In that case the data isn't in the file and no tool can recover it. If you still have the no-audio temp file from the same generation, that one will usually work.
