Workflow Rescue

Recover ComfyUI workflows from your videos.

Extracts the embedded workflow JSON from MP4, MOV, WebM, and MKV files exported by ComfyUI-VideoHelperSuite. Everything runs locally in your browser — files never leave your computer.

What this is for

If a video made with ComfyUI-VideoHelperSuite no longer drag-drops into ComfyUI, the workflow data may still be embedded in the file in a shape the frontend can't read directly. This page extracts it and gives you a workflow.json file you can open with ComfyUI's normal Load button.

If you get "no workflow found"

For videos that include audio, the workflow may have been stripped at write time — see VHS PR #653. In that case the data isn't in the file and no tool can recover it. If you still have the no-audio temp file from the same generation, that one usually works.
